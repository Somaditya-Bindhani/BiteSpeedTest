# Gamma4 Backend

## Description

This is backend repo Bitespeed Backend Task Identity Reconciliation.

## Cloning

```bash
git clone https://github.com/Somaditya-Bindhani/BiteSpeedTest.git
cd BiteSpeedTest
```

## Run

```bash
#it will create a new postgres db and run the server in PORT 5000
docker-compose up

```
